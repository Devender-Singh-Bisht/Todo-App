# 📝 Todo App

A simple, lightweight Todo web app built with HTML, CSS, and JavaScript. Allows users to add, delete, and manage daily tasks efficiently.

## 🚀 Features
- Add and delete tasks
- Data saved in `localStorage`
- Clean and minimal UI

## 🔧 Tech Stack
- HTML
- CSS
- JavaScript
